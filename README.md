# ⚡ Custom RISC-V Soft-Core Processor (Work In Progress)

## 📌 Project Overview
Designing and implementing a custom RISC-V soft-core processor from scratch. The focus is on understanding computer architecture fundamentals by translating the RISC-V ISA (Instruction Set Architecture) into functional digital logic optimized for deployment on FPGA hardware.

## 🛠️ Tech Stack & Tools
* **Languages:** SystemVerilog / Verilog
* **Tools:** Vivado, EDA Playground
* **Concepts:** CPU Architecture, Pipelining, Instruction Decoding, RTL Design
